# 养老金计算详细案例演示

## 案例背景

**李先生的基本情况：**
- 👨 性别：男职工
- 📅 出生日期：1979年2月
- 🏢 工作经历：原在职员工，2025年失业后转为灵活就业
- 💰 当前状态：失业后作为灵活就业人员继续缴纳养老保险

**历史缴费情况：**
- ✅ 已累计缴费：300个月（25年）
- 📊 历史平均缴费指数：1.0（一直按社平工资100%缴费）
- 💳 个人账户当前余额：200,000元

**当前时间：**
- 📆 2025年12月

**待决策问题：**
李先生想知道，选择哪个缴费档位继续缴费最合适？

---

## 第一步：查询法定退休年龄

### 1.1 基础信息
- 人员类型：男职工
- 出生年月：1979年2月（197902）

### 1.2 查询退休年龄区间表

男职工退休年龄区间表：
```
196501-196504 → 6001（60岁1个月）
196505-196508 → 6002（60岁2个月）
...
197001-197004 → 6104（61岁4个月）
197101-197104 → 6107（61岁7个月）
...
1977年及之后 → 6300（63岁）
```

### 1.3 计算结果

李先生出生于1979年2月，属于"1977年及之后"：
- **退休年龄代码：6300**
- **法定退休年龄：63岁**
- **退休时间：1979年2月 + 63岁 + 1个月 = 2042年3月**

**结论：李先生将在2042年3月退休，距今还有16年3个月。**

---

## 第二步：计算剩余缴费时间

### 2.1 时间计算

```
退休时间：2042年3月
当前时间：2025年12月

剩余缴费月数 = (2042 - 2025) × 12 + (3 - 12)
            = 17 × 12 - 9
            = 204 - 9
            = 195个月
            = 16年3个月
```

**结论：李先生还需缴费195个月。**

### 2.2 退休时累计缴费年限

```
历史缴费：300个月（25年）
未来缴费：195个月（16.25年）

总缴费月数 = 300 + 195 = 495个月
总缴费年限 = 495 ÷ 12 = 41.25年
```

**结论：李先生退休时累计缴费41.25年。**

---

## 第三步：计算计发月数N

### 3.1 精确退休年龄

```
退休年份：2042年
出生年份：1979年
退休月份：3月
出生月份：2月

精确年龄 = (2042 - 1979) + (3 - 2) / 12
        = 63 + 0.083
        = 63.08岁
        ≈ 63岁
```

### 3.2 查询N表

N值计发月数表（国家统一规定）：
```
60岁 → 93个月
61岁 → 86个月
62岁 → 79个月
63岁 → 73个月 ✓
```

**结论：李先生的计发月数N = 73个月。**

---

## 第四步：四档缴费方案对比

现在我们计算李先生选择不同档位的结果。北京2025年社平工资为12,049元。

### 方案1：基础档（60%）

#### 4.1.1 缴费基数
```
社平工资：12,049元
档位比例：60%
缴费基数 = 12,049 × 60% = 7,229元/月
```

#### 4.1.2 每月个人缴费（灵活就业20%）
```
每月缴费 = 7,229 × 20% = 1,445.80元
```

#### 4.1.3 未来个人总缴费
```
剩余月数：195个月
总缴费 = 1,445.80 × 195 = 281,931元
```

#### 4.1.4 个人账户增长

**每月计入个人账户（8%）：**
```
每月计入 = 7,229 × 8% = 578.32元
```

**未来195个月累计计入：**
```
新增账户 = 578.32 × 195 = 112,772元
```

**考虑利息（简化计算，假设平均存款时间为195/2个月）：**
```
平均年限 = 195 ÷ 12 ÷ 2 = 8.125年
利息因子 = 1 + 1.5% × 8.125 = 1.122

退休时账户余额 = (200,000 + 112,772) × 1.122
               = 312,772 × 1.122
               = 350,930元
```

#### 4.1.5 计算养老金

**A. 个人账户养老金：**
```
个人账户养老金 = 350,930 ÷ 73 = 4,807元/月
```

**B. 基础养老金：**

首先计算最终平均缴费指数：
```
历史指数：1.0
历史月数：300个月
未来指数：7,229 ÷ 12,049 = 0.6
未来月数：195个月

最终指数 = (1.0 × 300 + 0.6 × 195) ÷ (300 + 195)
        = (300 + 117) ÷ 495
        = 417 ÷ 495
        = 0.842
```

计算指数化月平均缴费工资：
```
指数化工资 = 0.842 × 12,049 = 10,145元
```

计算基础养老金：
```
基础养老金 = (社平工资 + 指数化工资) ÷ 2 × 缴费年限 × 1%
          = (12,049 + 10,145) ÷ 2 × 41.25 × 1%
          = 11,097 ÷ 2 × 41.25 × 0.01
          = 5,548.50 × 0.4125
          = 2,289元/月
```

**C. 总养老金：**
```
月养老金 = 基础养老金 + 个人账户养老金
        = 2,289 + 4,807
        = 7,096元/月
```

#### 4.1.6 方案1总结

| 项目 | 数值 |
|------|------|
| 缴费基数 | 7,229元/月 |
| 每月缴费（20%） | 1,446元 |
| 未来总缴费 | 281,931元 |
| 退休时账户余额 | 350,930元 |
| 月养老金 | 7,096元 |
| 基础养老金 | 2,289元 |
| 个人账户养老金 | 4,807元 |

### 方案2：标准档（100%）

#### 4.2.1 缴费基数
```
社平工资：12,049元
档位比例：100%
缴费基数 = 12,049元/月
```

#### 4.2.2 每月个人缴费
```
每月缴费 = 12,049 × 20% = 2,409.80元
```

#### 4.2.3 未来个人总缴费
```
总缴费 = 2,409.80 × 195 = 469,911元
```

#### 4.2.4 个人账户增长

**每月计入（8%）：**
```
每月计入 = 12,049 × 8% = 963.92元
```

**未来累计：**
```
新增账户 = 963.92 × 195 = 187,964元
```

**考虑利息：**
```
利息因子 = 1.122（同上）
退休时账户 = (200,000 + 187,964) × 1.122
          = 387,964 × 1.122
          = 435,259元
```

#### 4.2.5 计算养老金

**A. 个人账户养老金：**
```
个人账户养老金 = 435,259 ÷ 73 = 5,962元/月
```

**B. 基础养老金：**

最终平均缴费指数：
```
未来指数：12,049 ÷ 12,049 = 1.0

最终指数 = (1.0 × 300 + 1.0 × 195) ÷ 495
        = (300 + 195) ÷ 495
        = 495 ÷ 495
        = 1.0
```

指数化工资：
```
指数化工资 = 1.0 × 12,049 = 12,049元
```

基础养老金：
```
基础养老金 = (12,049 + 12,049) ÷ 2 × 41.25 × 1%
          = 12,049 × 41.25 × 0.01
          = 4,970元/月
```

**C. 总养老金：**
```
月养老金 = 4,970 + 5,962 = 10,932元/月
```

#### 4.2.6 与基础档对比

```
多缴费用 = 469,911 - 281,931 = 187,980元
每月多领 = 10,932 - 7,096 = 3,836元
回本时间 = 187,980 ÷ 3,836 = 49.0个月 ≈ 4年1个月
```

#### 4.2.7 方案2总结

| 项目 | 数值 | vs基础档 |
|------|------|----------|
| 缴费基数 | 12,049元/月 | +66.7% |
| 每月缴费 | 2,410元 | +964元 |
| 未来总缴费 | 469,911元 | +187,980元 |
| 月养老金 | 10,932元 | +3,836元 |

## 第六步：长期收益分析

假设李先生在2042年3月退休（63岁），活到85岁，共领取22年（264个月）养老金。

### 6.1 基础档（60%）

```
总缴费：281,931元
累计养老金：7,096 × 264 = 1,873,344元
净收益：1,873,344 - 281,931 = 1,591,413元
投资回报率：1,591,413 ÷ 281,931 × 100% = 564.5%
```

### 6.2 标准档（100%）

```
总缴费：469,911元
累计养老金：10,932 × 264 = 2,886,048元
净收益：2,886,048 - 469,911 = 2,416,137元
投资回报率：2,416,137 ÷ 469,911 × 100% = 514.2%
```

### 6.3 进阶档（200%）

```
总缴费：939,822元
累计养老金：14,801 × 264 = 3,907,464元
净收益：3,907,464 - 939,822 = 2,967,642元
投资回报率：2,967,642 ÷ 939,822 × 100% = 315.7%
```

### 6.4 尊享档（300%）

```
总缴费：1,396,629元
累计养老金：18,565 × 264 = 4,901,160元
净收益：4,901,160 - 1,396,629 = 3,504,531元
投资回报率：3,504,531 ÷ 1,396,629 × 100% = 250.9%
```

### 6.5 长期收益对比表

|------|--------|-----------|--------|--------|
| 基础60% | 28万 | 187万 | **159万** | **564.5%** |

**关键发现：**
- 💰 **绝对收益**：档位越高，净收益越大
- 📊 **回报率**：档位越低，回报率越高

## 第七步：给李先生的建议

### 情况1：经济宽裕，身体健康

**推荐：标准档或进阶档**

理由：
- ✅ 回本快（4-7年）
- ✅ 月养老金显著提高
- ✅ 长期总收益最大
- ⚠️ 需要承担较高月缴费

### 情况2：收入一般，求稳健

**推荐：基础档**

理由：
- ✅ 月缴费最低（1,446元）
- ✅ 回报率最高（646%）
- ✅ 资金压力小
- ⚠️ 退休后月养老金较低

### 情况3：收入波动，灵活调整

**建议：根据年收入灵活选择**

- 收入好时：选高档
- 收入差时：选低档
- 每年可调整一次

## 附录：关键公式汇总

### 1. 退休时间
```
退休年月 = 出生年月 + 法定退休年龄 + 1个月
```

### 2. 剩余缴费月数
```
剩余月数 = (退休年 - 当前年) × 12 + (退休月 - 当前月)
```

### 3. 个人账户余额
```
退休时账户 = (历史余额 + 新增缴费 × 8% × 剩余月数) × 利息因子
利息因子 = 1 + 年利率 × (剩余年数 ÷ 2)
```

### 4. 最终缴费指数
```
最终指数 = (历史指数 × 历史月数 + 未来指数 × 未来月数) ÷ 总月数
未来指数 = 缴费基数 ÷ 社平工资
```

### 5. 基础养老金
```
基础养老金 = (社平工资 + 指数化工资) ÷ 2 × 缴费年限 × 1%
指数化工资 = 最终指数 × 社平工资
```

### 6. 个人账户养老金
```
个人账户养老金 = 退休时账户余额 ÷ 计发月数N
```

### 7. 月养老金
```
月养老金 = 基础养老金 + 个人账户养老金
```

### 8. 回本时间
```
回本月数 = (高档总缴费 - 低档总缴费) ÷ (高档月养老金 - 低档月养老金)
```
